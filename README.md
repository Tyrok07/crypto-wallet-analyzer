# Kripto Cüzdan Analiz: İşlem Tarzı ve Profil

Bu Streamlit app, public cüzdan adresini girerek:

- ne alıyor ne satıyor,
- işlem sıklığı,
- strateji tipi (meme, swing, HODL, bot),
- risk profili,
- PnL ve win rate,

gibi özellikleri **akıl yürütmeyle özetler**.

## Özellikler

- Çoklu ağ desteği: Ethereum, BSC, Polygon, Arbitrum, Optimism, Base, Avalanche, Solana, Tron, Bitcoin
- Swap işlemleri zaman serisi grafiği
- Token bakiye dağılımı grafiği
- PnL ve win rate özeti
- Strateji ve risk etiketi

## Kullanım

1. Bu repoyu GitHub'a yükleyin.
2. Streamlit Community Cloud'da bu repoyu seçin.
3. Main file olarak `app.py` seçin.
4. Deploy edin.

URL örneği: `https://<your-username>.streamlit.app`
